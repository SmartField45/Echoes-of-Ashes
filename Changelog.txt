🟢 PUBLIC CHANGELOG v0.6 Beta 2
Echoes of Ashes - Prototype Build v0.6 Beta 2

#=========================
⚔️ Combat System Overhaul
#=========================
1. Rebalanced bombs to prevent them from being overpowered.
2. Fixed an issue where bombs could deal multiple damage instances to the same target.
3. Fixed an issue where bomb explosion sound effects could play twice under certain conditions.

#=====================
👹 Enemy AI Evolution
#=====================
1. Added Tutorial Training Dummies for combat practice.

#=================
🎨 UI & Game Flow
#=================
1. Fixed an issue where the "Game Over" menu buttons were unclickable.
2. Fixed an issue where the "Pause Menu" buttons were unclickable.

#========================
🎒 Systems & Interaction
#========================
1. Completely redesigned the Tutorial World.
- Learn movement naturally through gameplay.
- Learn sprinting using quest progression.
- Learn inventory and item notifications.
- Learn Light Attack and Heavy Attack.
- Learn Dash and invincibility frames.
- Learn bomb throwing through environmental obstacles.
- Learn healing mechanics before entering Chapter 1.

#===================
💾 Save Persistence
#===================
1. Improved save reliability by locating the active World node instead of relying on the current scene.
2. Fixed an infinite respawn loop caused by loading player data with zero health.
3. Resetting save data now also clears QuestManager runtime data.
4. Tutorial items are now permanently removed after entering Chapter 1.
5. Improved synchronization between runtime data and save files.

#======================
🌍 World & Exploration
#======================
1. Added a brand-new story-driven Tutorial World featuring eight sequential tutorial stages.
